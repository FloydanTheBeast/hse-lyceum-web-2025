# JS

Строки

1. Напишите функции:
    - `compressString(s)` - принимет строку и заменет повторяющиеся символы количеством повторений
        > Считайте, что во входящей строке не бывает цифр
    - `decompressString(s)` - совершает обратное преобразование

```js
compressString('АБВ') // АБВ
compressString('АААБББВВВ!!!') // А3Б3В3!3
compressString('ОККККО') // О4КО

decompressString('А3Б3В3!3') // АААБББВВВ!!!
```

# JS

Строки

2. Напишите функцию `atoi(s)` (P.S. `atoi` - функция стандартной библиотеки C++, **конвертирующая строку в 32-битные число**)
    1. Строка интерпретируется слева-направо
    2. Пробелы игнорируются
    3. Знак числа определяется символом "-" или "+", по умолчанию оно считается положительным
    4. Обнаружение символа, отличающегося от пробела или цифры, останавливает перевод
    5. Повторное обнаружение символов "+" и "-" также останавливает перевод
    6. При отсутствии в строке цифр функция возвращает 0
    7. Если результат выходит за пределы отрезка $[-2^31, 2^31 - 1]$, возвращаемое функцией значение всё равно должно входить в отрезок

```js
atoi("42") // 42
atoi("     +42    ") // 42
atoi("  ++42  ") // 0
atoi("4abc56") // 4
atoi("abc456") // 0
atoi("999999999999") // 2147483647 (2^31 - 1)
```

# JS

Рекурсивные функции

1. Напишите функцию `fibonacci(n)`, возвращающую `n`-ый элемент последовательности [чисел Фибоначчи](https://ru.wikipedia.org/wiki/%D0%A7%D0%B8%D1%81%D0%BB%D0%B0_%D0%A4%D0%B8%D0%B1%D0%BE%D0%BD%D0%B0%D1%87%D1%87%D0%B8)

2. Напишите функцию `generateValidParenthesesOfLength(n)`, которая сгенерирует массив строк из всех возможных комбинаций правильных скобочных последовательностей, которые можно составить из `n` пар скобок
    - Считаем что скобки только круглые
    - $0 < n < 30$
    > P.S. **Правильная скобочная последовательность** - такая последовательность скобок, у которой для каждой открывающей скобки есть соответствующая ей закрывающая
    
```js
generateValidParenthesesOfLength(1)
// ['()']
generateValidParenthesesOfLength(2)
// ['()()', '(())']
generateValidParenthesesOfLength(3)
// ['()()()', '(())()', '()(())', '(()())', '((()))']
```

# JS

Массивы

1. Напишите функцию `deduplicate(arr)`, принимающую массив чисел и возвращающую новый массив, содержащий только уникальные элементы, отсортированные по возрастанию
    ```js
    deduplicate([1, 2, 2, 3, 4, 4, 5]) // 1 2 3 4 5
    deduplicate([2, 5, 8, 2, 1, 10, 100, 4, 8]) // 1 2 4 5 8 10 100
    ```
2. Напишите функцию `findSecondAbsMax`, возвращающую второе по величине модуля уникальное число
    ```js
    findSecondMax([1, 2, 3, 4, 5]) // 4
    findSecondMax([10, 5, 20, 8, 20, 3]) // 10
    findSecondMax([2, 100, -74, 8, -5, 54]) // -74
    ```
3. Добавьте **метод массива** `arr.sqrt()`, возвращающий новый массив, где каждый числовой элемент возведён в квадрат
    > Если массив содержит нечисловые значения, то выбросите ошибку
    
    ```js
    [1, 2, 3, 4, 5].sqrt() // [1, 4, 9, 16, 25]
    [1, 2, 3, 4, "Строка"].sqrt() // Error
    ```

# JS

Объекты

1. Напишите функцию `inverseObject(obj)`, которая принимает объект и меняет ключи и значения местами.
    > Если есть совпадающие значения, то ключи нужно записать в массив

    <br />

    ```js
    inverseObj({ a: 1, b: 2, c: 3 }) // { 1: "a", 2: "b", 3: "c" }
    inverseObj({ a: 1, b: 2, c: 2, d: 3, e: 1 }) // { 1: ["a", "e"], 2: ["b", "c"], 3: "e" }
    ```
2. Напишите функцию `mergeRight(obj1, obj2)`, выполняющую слияние двух объектов и отдающее приоритет второму объекту при наличии одинаковых ключей
    ```js
    mergeRight({ a: 1, b: 2 }, { b: 10, c: 3 }) // { a: 1, b: 10, c: 3 }
    ```
